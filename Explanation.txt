Given an array of strings, consisting of only 0s and 1s. Also two integers n and k.
Now your task is to find the maximum number of strings that you can form with given n 0s and k 1s. Each 0 and 1 can be used at most once.

Sample input:
10 0 1
1 1

Sample output:
2

Explanation:
You can only form 0 and 1 from the given array. n 0's and k 1's

Sample input:
10 0001 111001 1 0
5 3

Sample Output:
4

Explanation:
n 0's and k 1's so, 10, 0001, 1, 0
so the output is 4
